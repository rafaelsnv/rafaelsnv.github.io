Sobre

O CinePortal é um portal de filmes desenvolvido como um trabalho para a disciplina Desenvolvimento de Interfaces WEB do curso de Sistemas de Informaçao na PUC-MG. Tem como base a framework Bootstrap e conhecimentos em HTML e CSS.
